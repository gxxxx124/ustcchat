# 大型粒子加速器技术知识库

## 1. 粒子加速器基本原理

### 1.1 加速器类型
- **直线加速器 (Linac)**: 粒子沿直线路径加速，适用于电子和质子
- **环形加速器 (Synchrotron)**: 粒子在环形轨道中加速，可达到更高能量
- **回旋加速器 (Cyclotron)**: 使用恒定磁场和交变电场加速粒子

### 1.2 加速原理
粒子加速器通过电磁场对带电粒子施加力，使其获得动能。主要机制包括：
- 电场加速：直接对粒子做功
- 磁场约束：控制粒子轨道
- 射频腔：提供交变电场加速

## 2. 大型加速器设施

### 2.1 欧洲核子研究中心 (CERN)
- **大型强子对撞机 (LHC)**: 27公里周长，质子能量7 TeV
- **LHCb实验**: 研究B介子物理
- **ATLAS和CMS探测器**: 发现希格斯玻色子

### 2.2 其他重要设施
- **费米实验室 (Fermilab)**: Tevatron对撞机
- **斯坦福直线加速器中心 (SLAC)**: 3公里直线加速器
- **日本高能加速器研究机构 (KEK)**: SuperKEKB对撞机

## 3. 关键技术系统

### 3.1 磁铁系统
- **二极磁铁**: 弯曲粒子轨道
- **四极磁铁**: 聚焦束流
- **六极磁铁**: 校正色散
- **超导磁铁**: 产生强磁场，降低功耗

### 3.2 真空系统
- **超高真空**: 压力 < 10^-9 mbar
- **束流管道**: 不锈钢或铝制，内壁特殊处理
- **真空泵**: 离子泵、钛升华泵、低温泵

### 3.3 射频系统
- **射频腔**: 提供加速电场
- **功率源**: 高功率射频放大器
- **频率**: 通常为几百MHz到几GHz

## 4. 束流动力学

### 4.1 束流参数
- **能量**: 通常以GeV或TeV表示
- **流强**: 粒子数/秒
- **亮度**: 碰撞率指标
- **发射度**: 束流质量参数

### 4.2 束流控制
- **束流位置监测**: 束流位置探测器
- **束流尺寸测量**: 束流轮廓监测
- **束流损失监测**: 辐射监测系统

## 5. 辐射防护

### 5.1 辐射类型
- **同步辐射**: 弯曲轨道产生
- **轫致辐射**: 电子减速产生
- **中子辐射**: 高能碰撞产生
- **伽马辐射**: 各种衰变过程

### 5.2 防护措施
- **屏蔽设计**: 混凝土、铅、铁屏蔽
- **辐射监测**: 实时剂量监测
- **安全联锁**: 自动保护系统
- **人员培训**: 辐射安全培训
